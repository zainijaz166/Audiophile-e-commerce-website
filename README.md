# Audiophile E-commerce Website - Frontend Mentor Solution

This is my solution to the [Audiophile e-commerce website challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/audiophile-ecommerce-website-C1qS_QpS3). This project is a multi-page e-commerce platform built using only core web technologies, focusing on high-quality UI and responsive layouts.

## Features

* **Responsive Multi-Page Layout:** A fully responsive store with dedicated pages for Headphones, Speakers, Earphones, and individual product details.
* **Mobile-First Design:** Developed using a mobile-first approach, ensuring a seamless experience across mobile, tablet, and desktop.
* **Custom Form Implementation:** A professional Checkout page featuring billing details, shipping info, and custom radio buttons for payment methods.
* **Clean Semantic HTML:** Built with accessible tags like `<nav>`, `<main>`, `<section>`, and `<footer>` for better SEO and code structure.

## Technologies Used

* **HTML5:** Semantic markup for structured content.
* **CSS3:** Custom Variables (colors/fonts), Flexbox, CSS Grid (for forms), and Media Queries.
* **Google Fonts:** Integrated "Manrope" font family for premium typography.
* **Git & GitHub:** Used for version control and project hosting.

## Project Highlights

In this project, I successfully managed a complex file structure with over 10 separate HTML pages. Key technical challenges I solved included:
1.  **Dynamic Layout Swapping:** Implementing CSS logic to alternate image and text positions using Flexbox.
2.  **Asset Management:** Handling a large library of assets for different device resolutions (Desktop, Tablet, Mobile).
3.  **Responsive Navigation:** Creating a modular navbar and footer that stays consistent across all pages.
